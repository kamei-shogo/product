# 大学自動ログイン Chrome拡張機能

## 概要

このプログラムは、大学のログイン画面（大阪大学IT認証基盤サービス）に自動でログインするためのChrome拡張機能です。

## ファイル構成

| ファイル名 | 説明 |
|---|---|
| `manifest.json` | Chrome拡張機能の設定ファイル |
| `content.js` | ログインページにIDとパスワードを自動入力し、ログインボタンをクリックするスクリプト |

## 使い方

1. `content.js` をテキストエディタで開きます。
2. 以下の箇所に自分のIDとパスワードを入力します。

    ```js
    const userId = "ここにID";       // 自分のIDに変更
    const password = "ここにパスワード"; // 自分のパスワードに変更
    ```

3. Chromeのアドレスバーに `chrome://extensions/` と入力して拡張機能の管理ページを開きます。
4. 右上の「デベロッパーモード」をオンにします。
5. 「パッケージ化されていない拡張機能を読み込む」をクリックし、このフォルダを選択します。
6. 大学のログインページにアクセスすると、自動的にIDとパスワードが入力されてログインされます。

## 注意事項

- IDとパスワードは初期状態では空欄（プレースホルダーテキスト）になっています。実際に使用する際は、必ず自分のIDとパスワードに書き換えてください。
- このファイルをそのままGitHubなどに公開する場合は、IDとパスワードを入力したままコミットしないよう注意してください。
