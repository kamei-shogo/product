# 大学自動ログイン Chrome拡張機能

## 概要

このプログラムは、大阪大学IT認証基盤サービス（CLE）へのログインを自動化するChrome拡張機能です。
以下の3ステップを自動で処理します。

1. **SAMLログインボタンのクリック** — ログイントップ画面で「ログイン(大阪大学個人IDを持っている方）」ボタンを自動クリックします。
2. **IDとパスワードの入力・送信** — 認証フォームに自分のIDとパスワードを自動入力し、ログインボタンをクリックします。
3. **ロール選択** — ロール選択画面で最後の選択肢を選び、OKボタンをクリックします。

## ファイル構成

| ファイル名 | 説明 |
|---|---|
| `manifest.json` | Chrome拡張機能の設定ファイル。対象URLやスクリプトの読み込みを定義します。 |
| `content.js` | 上記3ステップの自動操作を行うスクリプト |

## 使い方

1. `content.js` をテキストエディタで開きます。
2. 以下の箇所に自分のIDとパスワードを入力します。

    ```js
    const userId = 'ここにIDを入力';
    const password = 'ここにパスワードを入力';
    ```

3. Chromeのアドレスバーに `chrome://extensions/` と入力して拡張機能の管理ページを開きます。
4. 右上の「デベロッパーモード」をオンにします。
5. 「パッケージ化されていない拡張機能を読み込む」をクリックし、このフォルダを選択します。
6. CLEのページにアクセスすると、ログインが自動的に完了します。

## 注意事項

- IDとパスワードは初期状態ではプレースホルダーテキストになっています。実際に使用する際は、必ず自分のIDとパスワードに書き換えてください。
- このファイルをそのままGitHubなどに公開する場合は、IDとパスワードを入力したままコミットしないよう注意してください。
